Адаптивний дизайн Адаптивний дизайн - це конфігурація, в якій сервер відправляє однаковий HTML-код на будь-які пристрої, а розміри та положення елементів сторінки коригуються за допомогою CSS. 2. Використання метатега viewport Щоб повідомити браузеру, що веб-сторінка адаптується до будь-яких пристроїв, додайте в заголовок документа такий метатег: 3. Медіа запити Правило @media дозволяє вказати тип та характеристики носія, для якого буде застосовано вказаний стиль (принтер, монітор та ін.). Медіа-функції задають технічні характеристики пристрою, на якому відображається документ. Більшість функцій містять приставку min- і max-, які відповідають мінімальному і максимальному значенню. Так, max-width: 400px означає, що ширина вікна браузера менше 400 пікселів, а min-width: 1000px, навпаки, повідомляє, що ширина вікна понад 1000 пікселів. Стиль виконується в тому випадку, якщо запит повертає істину. Приклад: @media screen and (max-width: 500px) - описаний далі стиль буде застосований до сторінки, яка переглядається на екрані шириною не більше 500 пікселів. @media screen and (max-width: 500px) { .phone { margin: auto; } } @media screen and (min-width: 500px) { .phone { float: right; } } 5. Елементи навігації розміщено поруч, без маркерів, без підкреслення, з відступами між елементами по горизонталі (стале значення). - Весь заголовок оформити як флекс-контейнер.
